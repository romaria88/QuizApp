Приложение Quiz App

1. Создать разметку в index.html 
2. Добавить стили в style.css
3. Подключите внешний шрифт
4. Приложение должно быть размещено по центру экрана - компактно
5. В файле js есть массив объектов quizData. Данный массив нужно перебрать. Когда пользователь ответит на все вопросы, ему должен выводиться результат и дальше при нажатии на кнопку reload список вопрос загружается заново и пользователь может пройти тест еще раз.
